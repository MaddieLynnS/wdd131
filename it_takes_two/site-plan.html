Overview

Purpose

The purpose of this site is to provide a fun, ad-free place for kids to play simple games and get to know each other better. This will be accomplished through a variety of simple two-player games that are interspersed with questions that the players can then discuss in between turns to learn more about the other player.

Audience

My site audience is for younger kids ages 7-13 who often enjoy playing computer games but need something simple.

Dynamic elements

Javascript will do most of the heavy lifting for this site. There will be a simple search bar for finding games, which will filter out what games are shown. Javascript will also store the logic for the games and determine which game is being played. It will also select from the JSON file of different questions based on which category the players selected at the beginning of the game and pick one to show to the players in between turns.
